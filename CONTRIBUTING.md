# Contributing

이 레포는 Hermes Agent에서 OpenAI Codex provider 사용 중 발생한 `NoneType` 오류를 설명하기 위한 공개 문서 레포입니다.

오탈자 수정, 추가 재현 사례, upstream 패치 정보, 버전별 차이점 제보를 환영합니다.

## 수정 제안 방법

1. Issue를 생성하거나
2. Pull Request로 Markdown 문서를 수정해 주세요.

## 문서 작성 기준

- 가능한 한 초보자도 이해할 수 있게 설명합니다.
- 실제 로그/토큰/인증 정보는 절대 포함하지 않습니다.
- 재현 경로, 원인, 해결 방법을 분리해서 작성합니다.
